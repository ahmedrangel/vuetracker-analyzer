# VueTracker Analyzer

# Quick Setup
Install dependencies
```bash
pnpm i
```

Install puppeter browser
```bash
npx puppeteer browsers install chrome
```

Create env secrets file
```md
NUXT_ANALYZER_PROXY_URL=
NUXT_ANALYZER_PROXY_OS=
```

Start dev
```bash
pnpm dev
```
